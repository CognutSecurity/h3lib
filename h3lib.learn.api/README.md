# Smart Engine - MyriadLobe
This is the main smart components for project MyraidHub, where we develop several different smart services using machine learning.

## Installation
In order to run the code in this project, you need install:
* Python 2.6+
* numpy
* scipy
* scikit-learn
* matplotlib
* nltk 3.0+

You probably wanna try:
* PyCharm IDE for python
* Jupyter notebook for fast scripting in web browser

In order to run some algorithms, you also need to download dictionaries for NLTK, in terminal:
```bash
python -c "import nltk; nltk.download(['punkt', 'vader_lexicon'])"
```
All above can be easiliy installed using [pip](https://pypi.python.org/pypi/pip).
